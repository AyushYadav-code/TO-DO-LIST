To-Do List Application :
A simple console-based To-Do List application built using Python. This project allows users to add, view, and remove tasks while storing them in a text file so that tasks persist even after closing the program.

Features :
- View all tasks
- Add new tasks
- Remove existing tasks
- Save tasks to a text file
- Load saved tasks automatically when the program starts
- User-friendly command-line interface

Technologies Used :
- Python 3
- File Handling
- Lists
- Loops and Conditional Statements
- Exception Handling
